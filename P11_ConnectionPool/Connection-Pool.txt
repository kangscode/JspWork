

# Connection Pool
- 데이터베이스에 연결하기 위한 Connection 객체입니다
  Connection 객체를 생성하고 해제하는 동안 시스템 자원이 소모됩니다
  이 문제를 해결하기 위해서 Connection Pools에 커넥션 객체들을 만들어 놓은 후에,
  커넥션 객체가 필요한 경우 객체를 할당해 주고, 사용이 끝나면 다시 커넥션 풀로 회수하는 방법입니다

1. jar 파일 추가 : WEB-INF -> lib
- tomcat server 설치 폴더안에 기본 제공합니다
  > lib -> tomcat-dbcp.jar
  
2. DBDP 정보 설정
- .xml
  > 웹이나 앱에서 데이터 및 구조화된 문서들을 위한 표준이고, 설정할 때 사용합니다
- context.xml : Connection Pool의 설정 파일
  > META-INF 안에 만듭니다
