# JSP 내장 객체 ( Implicit Object )
- 웹 컨테이너는 JSP 페이지에 상황에 따라서 필수적으로 사용되는 9개의 객체를 객체 생성없이 바로 사용할 수 있게 제공합니다

1. requsest
- 웹 브라우저의 요청 정보를 저장하고 있는 객체입니다

2. response
- 웹 브라우저의 요청에 대한 응답 정보를 저장하고 있는 객체입니다

3. out
- JSP 페이지의 출력 할 내용을 가지고 있는 출력 스트림 객체입니다

4. session
- 하나의 웹 브라우저 내에서 정보를 유지하기 위한 세션 정보를 저장하고 있는 객체입니다

5. application
- 웹 어플리케이션 Context의 정보를 저장하고 있는 객체입니다

6. pageContext
- JSP 페이지에 대한 정보를 저장하고 있는 객체입니다

7. page
- JSP 페이지를 구현한 자바 클래스 객체입니다

8. config
- JSP 페이지에 대한 설정정보를 저장하고 있는 객체입니다

9. exception
- JSP 페이지에서 예외가 발생한 경우에 사용되는 객체입니다
