

# JSP (Java Server Page)
- HTML 문서 안에 Java 코드를 삽입해서 웹 서버에 동적으로 웹페이지를 생성하여 클라이언트에 전달합니다

# Servlet 변환 과정
- *.jsp   ->   *.java   ->   *.class   ->   servlet 실행
	 jsp 파일을 컨테이너가
	 서블릿  .java로 변환
	 
# JSP 지시자
- 페이지가 실행 될 때 필요한 정보를 정의하는 역할입니다
- page 지시자
  > 해당 페이지에 대한 정보 및 설정을 하는 지시자 입니다
  include 지시자
  > 현재 페이지에 포함될 코드나 문서를 정의합니다
  taglib 지시자
  > JSP 페이지 내에서 사용되는 별도의 표현언어를 사용하기 위해 쓰이는 지시자 입니다
