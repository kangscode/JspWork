
# EL ( Expression Language ) : 표현 언어
- Java 코드가 들어있는 표현식을 좀 더 편리하게 사용하기 위한 데이터 출력 기능입니다

# EL 특징
- 표현식보다 편리하게 값을 출력할 수 있고, 변수와 여러가지 연산자도 적용할 수 있습니다
- JSP 내장객체에 저장된 속성 및 자바빈의 속성도 표현 언어에서 출력할 수 있습니다

# EL 형식
- ${ 표현식 or 값 }

# EL 에서 제공하는 내장객체
- scope
  > pageScope 		 => jsp page
    requestScope 	 => jsp request
    sessionScope 	 => jsp session
    applicationScope => jsp application

- 요청 매개변수
  > param
    : request.getParameter() 메서드를 사용한 것과 같습니다
    paramValues
    : request.getParameterValues() 메서드를 사용한 것과 같습니다
